# Python-Phishing-Simulation
A Flask web app for educational purposes: it simulates a VK (VKontakte) login page, intercepts logins, passwords and 2FA codes (saves them to log.txt), then redirects to vk.com. Used to study web security and phishing principles.

# ⚠️ Educational Purpose Only

This project is created **exclusively for educational and research purposes**.
It demonstrates how phishing attacks work to help developers and security researchers understand the risks.
